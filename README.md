# Project Overview

This project implements a cloud-hosted Linux web server using Infrastructure as a Service (IaaS) through Amazon Web Services EC2. The objective of the project is to demonstrate practical server administration, Linux command-line usage, DNS configuration, SSL/TLS implementation, and web hosting technologies.

The server hosts a deployment handbook and monitoring platform documenting the process of configuring a cloud-based web server from beginning to end.

The project demonstrates:

- Linux server administration
- Apache web hosting
- SSH remote management
- DNS configuration
- SSL/TLS implementation
- Bash scripting
- Cloud infrastructure deployment
